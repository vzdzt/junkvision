## ğŸš€ Getting Started

### Prerequisites
- Node.js (>=14)
- npm or yarn
- Expo CLI
- iOS Simulator (macOS) or Android Emulator

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/vzdzt/junkvision.git
   cd junkvision
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Run on specific platform**
   ```bash
   # iOS Simulator
   npm run ios

   # Android Emulator  
   npm run android

   # Web Browser
   npm run web
   ```
